# Amazon E-commerce Sales Analysis
This repo contains a beginner-friendly sales analysis project done in Powerbi.

## Tools Used
-Power Query Editor
-Powerbi

## Project Overview
This project analyzes the sales performance of Amazon.
It explores trends, product ratings and customer reviews using interactive dashboards.

## What I Learned
-How to clean messy data using formulas
-How to summarize KPIs using charts
-How to visualize insights in PowerBi.
